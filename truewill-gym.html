WEEK 1
▸ HOE HOSTEN OP GITHUB (IPHONE-BOOKMARK)
uitklappen
1
Ga naar github.com en maak een gratis account aan als je er nog geen hebt.
2
Klik rechtsboven op + → New repository. Naam: gym. Zet op Public. Vink Add a README file aan. Klik Create repository.
3
Klik op Add file → Upload files. Sleep truewill-gym.html erin. Klik Commit changes.
4
Ga naar Settings → Pages. Onder Branch: selecteer main → Save. Wacht 1-2 minuten.
5
URL wordt: https://JOUWUSERNAME.github.io/gym/truewill-gym.html
6
iPhone: open URL in Safari → deel-icoon → Zet op beginscherm. Beide gebruikers openen dezelfde URL — logs zijn gescheiden per profiel.
Logs worden opgeslagen in localStorage van Safari op het toestel. Gauthier en Mathieu hebben elk hun eigen gescheiden opslag.
